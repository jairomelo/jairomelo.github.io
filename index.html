¡Hola!
Este sitio es mi página personal. En primer lugar es un espacio para compartir parte de mi trabajo y experiencias. Básicamente es un reto personal para construir un sitio funcional, atractivo, personalizado, y que pueda en un momento ser de utilidad para otros.

La idea de construir el sitio desde 0 tiene como propósito documentar la construcción de un sitio " a la antigüita"... pero en UTF-8 :p . También es mi intento por evitar usar Jekyll en Github pages, por lo que es posible que en cualquier momento abandone este proyecto :'D

Este es el initial commit así que no esperes mucho de este espacio :D
